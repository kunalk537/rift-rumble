Welcome to Rift Rumble

Build, code, and battle with robots! Learn programming fundamentals and hardware integration through our interactive courses and hands-on projects. Each Rift Rumble kit provides students with a fully programmable and customizable robot that enables hands-on experimentation with competitive challenges to unlock core engineering skills.

Why Choose Rift Rumble?

Wi-Fi Connected

Each robot responds to Wi-Fi connection, enabling remote programming and real-time data collection.

Modular Design

LEGO-compatible chassis allows for endless customization and hardware modifications.

Dual Programming

Choose between block coding for beginners or Python for advanced users.

Advanced Sensors

Utilize a vast array of sensor inputs for complex robotics applications.
